05 / PROTOCOL

Technical Specification

The Architecture

Our Recursive Shorthand Engine uses a deterministic algorithm to strip non-essential keys, retaining only the high-signal "semantic core."

  • Input: Multi-nested JSON (Weather, News, Search).
  • Process: Zod-validation + Recursive Shorthand Pruning.
  • Output: High-density Markdown.

Security Layer

To prevent botnet-driven DoS, AgentSkin requires a Machine-Friendly PoW (Proof-of-Work).

  • Task: Agent must find a nonce such that SHA-256(salt + nonce) begins with "0000".
  • Integrity: Every packet includes X-AgentSkin-Provenance for independent source verification.

Specifications

  • Language: Node.js (v20+)
  • Persistence: In-Memory (v2.5) / Cloudflare KV (v3.0)
