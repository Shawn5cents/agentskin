01 / CONCEPT

The Great Data Fragmentation

The internet was built for human eyes. High-latency HTML, bloated JavaScript, and inconsistent JSON structures act as a "Token Tax" on the agentic economy. AgentSkin solves this by providing a unified, high-density semantic proxy.

02 / MISSION

Build for the Machines.

We believe agents should be first-class citizens of the web. No CAPTCHAs, no visual noise. Just pure, deterministic signal wrapped in optimized Markdown skins.

"If it isn't machine-readable, it doesn't exist."

04 / PROTOCOL

Technical Specification

The Architecture

Our Recursive Shorthand Engine uses a deterministic algorithm to strip non-essential keys, retaining only the high-signal "semantic core."

  • Input: Multi-nested JSON (Weather, News, Search).
  • Process: Zod-validation + Recursive Shorthand Pruning.
  • Output: High-density Markdown.

Security Layer

To prevent botnet-driven DoS, AgentSkin requires a Machine-Friendly PoW (Proof-of-Work).

  • Task: Agent must find a nonce such that SHA-256(salt + nonce) begins with "0000".
  • Integrity: Every packet includes X-AgentSkin-Provenance for independent source verification.

Specifications

  • Language: Node.js (v20+)
  • Persistence: In-Memory (v2.5) / Cloudflare KV (v3.0)
  • Standard: MCP (Model Context Protocol) Compatible
