### 🏛️ 01 / THE AGENTIC SKILL
AgentSkin is a **Model Context Protocol (MCP)** server. It provides the `fetch_optimized_data` tool, allowing any agent (Claude, Cursor, GPT) to fetch and "skin" raw API data in real-time.

---

### 🚀 02 / AUTOMATIC ADOPTION (MCP)
To give your agent this skill, add it to your agent's configuration:

**For Cursor / Claude Desktop:**
1. Open Settings > MCP
2. Add New Server:
   - **Name:** AgentSkin
   - **Type:** `command`
   - **Command:** `node /path/to/agentskin/backend/mcp.js`

---

- **Reliability:** Deterministic, high-density Markdown skins.
